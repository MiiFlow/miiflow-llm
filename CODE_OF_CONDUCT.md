# Code of Conduct

## Our Standards

We expect:
- Respectful communication
- Constructive feedback
- Focus on the code, not the person
- Help others learn

We don't accept:
- Harassment or abuse
- Discriminatory language
- Trolling or personal attacks
- Spam

## Scope

This applies to:
- GitHub issues and PRs
- Code reviews
- Community discussions
- Any project space

## Enforcement

Violations can be reported to: dev@miiflow.ai

We will:
- Review reports promptly
- Keep reporter identity confidential
- Take appropriate action

Actions may include:
- Warning
- Temporary ban
- Permanent ban
